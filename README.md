# LogAll

> Log all. Worry none.

Simple tax tracking for self-employed sole traders in the UK — cleaners, carers, gardeners, dog walkers, and more.

## Tech Stack

- React + Vite (PWA)
- Tailwind CSS
- React Router
- Supabase (auth + database)
- Cloudflare Pages (hosting)

## Getting started

```bash
npm install
cp .env.example .env.local
# Fill in your Supabase credentials
npm run dev
```

## Mileage rate

55p/mile (updated April 2026 — HMRC AMAP rate for cars and vans, including electric vehicles).
